# Two Talk: Concise Semantic Notation

**Two Talk** is a microlanguage for capturing ideas with speed and clarity in plaintext form. It uses **9 core symbols** that combine into **81 semantic primes**, which chain together with dots to create unlimited meaning. Think of it as Markdown for clear, compressed thought—or Toki Pona for precise notation.

Instead of sentences, Two Talk uses symbols to compress meaning in a way that is fast to write, easy to scan, and flexible across contexts. It's designed for quick note-taking, knowledge mapping, and everyday thinking.

---

## 9 Core Symbols

Each symbol represents a fundamental domain of meaning. When combined in pairs, they create the full semantic vocabulary.

| Symbol | Meaning | Category |
|--------|---------|----------|
| **@** | people / who | An |
| **#** | objects / what | Ob |
| **!** | value / quality | Ve |
| **?** | mind / thought | Mo |
| **>** | action / outward | Ak |
| **<** | relation / limit / inward | Li |
| **:** | time / when | Ti |
| **^** | space / where | Sa |
| **%** | scale / amount | Me |

## Chaining: Building Complex Meanings

Chaining combines multiple Two Talk primes to create nuanced meaning. A dot (·) joins one prime to the next—each addition narrows or intensifies the concept.

### Example 1: Simple Noun
**@#** → person (one prime, one root concept)

### Example 2: Qualified Concept
**@# · !@** → person + care = **caregiver**

### Example 3: Intensified Meaning
**@# · !@ · %!** → person + care + extreme = **devoted**

### Example 4: Technique
**#! · ?! · :<** → tool + know + past = **antiquated** (ancient or obsolete method)

## Key Rules for Two Talk

### Literals
- **Numbers** for amounts: 5, 2.5
- **Dates** in YYYY-MM-DD format: 2025-04-14
- **Times** in 24-hour clock: 1430
- **Names** as capitalized initials: BM for Barney Matthews

### Order
Write left to right in order of meaning: **who → what → work → when**

Example: **@# · ## · >! · :#** = person + thing + do + now = "the person is doing the thing now"

### Sentences
Each new line is a new sentence.

### Pronunciation
Each core symbol has a spoken name:
- @ (An), # (Ob), ! (Ve), ? (Mo), > (Ak), < (Li), : (Ti), ^ (Sa), % (Me)

Two Talk can be spoken aloud, which helps with learning.

## Real-World Examples

### Daily Note
**English:** I had a good conversation with my colleague about the project. It helped me understand the problem better.

**Two Talk:**
```
@! · >@ · <^ · @# · !!
>< · ?#
```

**Explanation:** @! (I/me), >@ (speech), <^ (with), @# (person), !! (good) + >< (transfer), ?# (perceive)

---

### Status Update
**English:** The bug from yesterday is fixed now, but there's new work coming tomorrow.

**Two Talk:**
```
## · !: · :<
%% · %> · <: · !! · :>
```

**Explanation:** ## (object), !: (bad/broken), :< (past), %> (many), %% (size), :> (future)

---

### Meeting Notes
**English:** We need to decide on the feature by Friday. The team has good ideas but needs more time to think.

**Two Talk:**
```
@% · !? · :#
## · !!
:: · %< · >: · !! · ?^
```

**Explanation:** @% (collective we), !? (need), :# (now) + ## (object), !! (good) + :: (time), %< (limited), >: (future), ?^ (focus)

## Getting Started: Four Steps

1. **Learn the 9 Matrix Symbols** — Master the base components representing primary domains: @ # ! ? > < : ^ %
2. **Build the 81 Core Primes** — Two base symbols intersect to establish standard primes (#! = tool, @# = person)
3. **Chain with Dots** — Connect root concepts to qualify thoughts on the fly: @# · >! · !!
4. **Log in Plaintext** — Deploy system shortcuts for quick notes, logs, or knowledge mapping
