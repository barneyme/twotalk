TWO TALK GUIDE


CORE SYMBOLS
------------
Each symbol represents a broad domain of experience depending on vector position.

@	 Subject: People (Who)                        State: Self (Internal)
#	 Subject: Object (What)                       State: Solid (Tangible)
!	 Subject: Value (Good)                        State: Max (Extra)
?	 Subject: Mind (Why)                          State: Hidden (Thought)
>	 Subject: Action (Push)                       State: Outward (Giving)
<	 Subject: Limit (Pull)                        State: Inward (Taking)
:	 Subject: Time (When)                         State: Now (Next)
^	 Subject: Space (Where)                       State: Here (Grounded)
%	 Subject: Scale (Status)                      State: More (Multiplied)
